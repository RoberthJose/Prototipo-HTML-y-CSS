HTML: La Estructura

¿Qué es HTML?

Lenguaje para estructurar el contenido de una página web, permitiendo organizar textos, imágenes y enlaces de manera clara y semántica.

Etiquetas Semánticas

Permiten organizar el contenido con significado, facilitando la lectura por personas y motores de búsqueda de manera eficiente y ordenada.

Header y Footer

Ubican encabezado y pie de página, separando secciones y ofreciendo una estructura clara para la navegación y presentación de información.

Main y Sections

Definen áreas principales y secciones, ayudando a distribuir el contenido de forma lógica y accesible dentro de la página web.

Article

Representa contenido independiente dentro de una sección, ideal para mostrar información completa y diferenciada de manera ordenada y comprensible.

CSS: El Diseño

¿Qué es CSS?

Lenguaje utilizado para dar estilo visual a las páginas, controlando colores, tipografías, tamaños y disposición de los elementos de manera elegante.

Colores y Tipografías

Mejoran la apariencia del sitio, transmitiendo identidad visual y haciendo que el contenido sea más legible y atractivo para los usuarios.

Espaciado

Controla márgenes, rellenos y separación de elementos, logrando un diseño equilibrado y estético sin sobrecargar ni saturar la página.

Bordes y Sombra

Ayudan a resaltar elementos importantes, dando profundidad y destacando secciones, botones o imágenes dentro del diseño general.

Layout con Flexbox

Permite dividir la página en columnas y filas fácilmente, facilitando la organización de contenidos y adaptando el diseño a diferentes tamaños de pantalla.

Prototipo Web

Navbar

Menú para navegar entre secciones de forma intuitiva, ofreciendo enlaces claros y accesibles que facilitan la interacción con la página.

Main de 3 Columnas

Organiza el contenido en bloques verticales, permitiendo una presentación clara y ordenada de los elementos principales del sitio web.

Artículos

Contienen información específica sobre temas concretos, distribuyendo contenido de manera organizada y visualmente atractiva para los usuarios.

Estética Visual

Uso de colores, sombras y tipografía que armonizan el diseño, generando una experiencia agradable y profesional al navegar por el prototipo.

Usabilidad

La interfaz debe ser fácil de entender y usar, priorizando claridad, accesibilidad y navegación sencilla para ofrecer una experiencia óptima.
